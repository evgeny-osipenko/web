class TestOrdering f => SingletonConstructor f Source #

This typeclass declares that a type constructor produces singletons. That is, for each possible type argument a, there is at most one possible object of type f a.

In other words, the instances of this class must satisfy the law:

testOrdering (fa :: f a) (fb :: f b) == EQI    ===>    a !~ b

Examples of good instances of this class are the singletons from GHC.TypeLits, and also TypeRep and SName.

There are no additional methods to implement in this class, it is only used as a marker.
